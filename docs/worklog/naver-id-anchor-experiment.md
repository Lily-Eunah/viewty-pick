## 결정 가이드
- **직접+링크 hit-rate 60% (높음)** → **B를 id 앵커링으로 재설계 권고**: 검색 결과에서 N 일치 항목을 채택해 큐레이션 정확 SKU 확보(변형/세트 오매칭 동시 해결). 제목 유사도는 보조.
- 링크 경유만 되는 비율 60% → 링크 해석 비용/차단 트레이드오프 고려.

## 상세 해석 + 권고 (큐레이션)

**핵심 결과**
- **직접 `item.productId == N` = 0/40.** 검색 API의 `productId`는 큐레이션 채널상품번호와 **다른 네임스페이스** → 직접 비교는 불가(이전 1차 점검과 일치). 앵커링은 **반드시 `item.link`의 `/products/{N}`** 로 해야 함.
- **링크 앵커 = 24/40 (60%).** brand/smartstore 67% · naver.me shortlink 46%(리다이렉트 해석 필요).
- **id없음 0** — 모든 큐레이션 URL에서 N 추출 성공(shortlink 13건 전부 리다이렉트로 해결).

**중요 단서: 링크 앵커 24건 중 ~13건이 운영자가 큐레이션한 *세트/다중팩* 페이지**
- 단품 hit(~11): #34 조선미녀 50ml · #70 에스네이처 300ml · #75 코스노리 150ml · #77 피지오겔 200ml 로션 · #79 듀이트리 60ml · #81 에스트라 `[싱글]` 40ml · #89 니들리 60g · #91 일리윤 250ml(17,600) · #96 파넬 15g · #258 유이크 18g · #98 VDL 13g.
- **세트 hit(~13): #50 메디큐브 `250ml×2` · #71 인터미션 `x2더블` · #76 닥터지 `1+1` · #82 아벤느 `+부스트` · #87 라운드랩 `3개` · #90 브링그린 `2개` · #94 아이소이 `본품+리필` · #95 아르마니 `리필위크` · #256 닥터지 `1+1` · #257 이즈앤트리 `더블구성` · #261 몽디에스 `1+1`.** → **운영자 naver URL이 세트 페이지를 가리킴**(단품가 표시 의도면 시트 URL을 단품으로 교체 필요 — catalog-match-map 시트정리에 추가).

**권고: 2-tier 하이브리드 매처로 재설계(별도 코드 PR)**
1. **Tier 1 — 링크 id 앵커(60%)**: 큐레이션 URL에서 N 추출(naver.me는 리다이렉트) → 검색 결과 중 `link`가 `/products/{N}`인 항목 채택 = **운영자 정확 SKU**. 변형/교차상품 오매칭 원천 제거(예: #77 피지오겔은 제목매칭이 `리페어 진정크림`(오매칭)을 잡았으나, 앵커는 정답 `레드수딩 AI 페이셜 로션 200ml`을 잡음 → 단순 제외보다 우월).
   - 앵커된 항목에도 `classifyOfferComposition` 적용: 단품 → 사용 / 세트·다중팩 → 개당가 or 제외(trust-first). 즉 **세트 URL은 정확히 식별해 일관 처리** + 시트정리 신호로 노출.
2. **Tier 2 — 폴백(40% miss)**: 현행 공식몰 + 단품우선 + 변형/폼 토큰 제목매칭 유지(랑콤·유세린·토리든·코스알엑스 등 N 미노출 제품).

**미스 40%(16건) 사유**: 큐레이션 N이 검색 100건에 안 뜸 — 결과수 0(브랜드스토어 미인덱스: #29·#30·#72), 또는 N이 가격비교/다른 채널에만 존재. 일부는 제목매칭이 대신 동작(#73 에뛰드 등). 따라서 앵커는 **부분 적용 + 제목매칭 폴백** 구조가 적절.

**주의/트레이드오프**
- naver.me shortlink 앵커(46%)는 **리다이렉트 1회/listing** 비용 — sync 시 호출량↑·차단 리스크. 캐시/배치 필요.
- 앵커는 정확 SKU를 주지만, 그 SKU가 세트면 가격 커버리지는 여전히 trust-first로 제한됨(틀린 단품가 대신 정확 세트 인식). 커버리지는 **시트 URL 단품화(A)** 로 회복.
