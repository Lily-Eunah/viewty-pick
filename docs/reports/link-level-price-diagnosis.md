## D. 매처 개선 제안 (바이오힐보發 — outlier 제거)
바이오힐보 올리브영 후보: **157,700**(`[슈링크홈디바이스] …세럼 30ml 기획` = 디바이스 번들) vs **39,000**(`[광채세럼] …세럼 30ml [단품/기획]` = 단품). 매처가 둘을 "모호"로 보류 → 둘 다 누락.

두 개선(상호보완):
1. **이종/번들 키워드 제외** — 슈링크·디바이스·홈디바이스·기획세트 등 device/번들 구성은 단품이 아니므로 후보 제외.
2. **크로스셀러 가격 outlier 제외(운영자 제안)** — 후보가 여럿이면 다른 판매처/후보 가격대(중앙값 등)에서 비정상적으로 동떨어진 값을 버리고 근접값 선택. (신뢰 참조가격 ≥1 필요.)

→ 둘 다 바이오힐보를 **39,000 단품**으로 자동 매칭. **즉시 조치 = manual_override 39,000**, 구조 개선 = 위 2개 매처 PR.
